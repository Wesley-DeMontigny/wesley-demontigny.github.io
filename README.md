## Software & Tools

### Smc-CAT 
A C++ sequential Monte Carlo sampler for the CAT Model (an important infinite mixture Bayesian phylogenetic model).
GitHub: https://github.com/Wesley-DeMontigny/Smc-CAT

### Phylogenetic Occupancy Modeling
A statistical model for incomplete genomic presence/absence data written with NumPyro and JAX, allowing for efficient multiple imputant of complete genomic content.
GitHub: https://github.com/Wesley-DeMontigny/Phylogenetic-Occupancy-Model
